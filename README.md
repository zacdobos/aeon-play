# AEON — play in your browser

**▶ Play: https://zacdobos.github.io/aeon-play/**

A historical grand strategy web game (EU4 × Civ × AoE2): start anywhere in history,
play any of ~190 real nations, forever. Everything runs in your browser — saves live
in localStorage.

This repository holds only the built site. The source lives in a private repository;
this deployment is updated with each release.
